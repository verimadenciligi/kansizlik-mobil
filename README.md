## Kullanım
### İndirme

```bash
git clone https://github.com/verimadenciligi/kansizlik-mobil.git
cd kansizlik-mobil
```

### Kurulum

Bağımlılıkları yükle

```bash
npm install
```

### Expo Go Uygulaması Üzerinde Çalıştırma

```bash
npm start
```

### IOS Simülatorünü Çalıştırma

```bash
npm run ios
```

### Android Simülatorünü Çalıştırma

```bash
npm run android
```
